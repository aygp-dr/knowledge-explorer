# knowledge-explorer

Tree-based knowledge navigation. Terminal UI built with [Bubble Tea](https://github.com/charmbracelet/bubbletea).

## Run

```bash
go run .
```

## Build

```bash
make build
./bin/knowledge-explorer
```

## Controls

- `j/k`: navigate
- `Enter`: select/details
- `q`: quit

#+TITLE: knowledge-explorer
#+AUTHOR: aygp-dr

Tree-based knowledge navigation. Terminal UI built with [[https://github.com/charmbracelet/bubbletea][Bubble Tea]].

** Run

#+begin_src bash
go run .
#+end_src

** Build

#+begin_src bash
make build
./bin/knowledge-explorer
#+end_src

** Controls

- =j/k=: navigate
- =Enter=: select/details
- =q=: quit
